## raspberry pi 센서 제어 코드 및 회로
### tensorflow lite 설치법
  $ python3 -m pip install tflite-runtime
### numpy 설치법
  $ pip install numpy
### numpy 버전 업그레이드
  $ pip install --upgrade numpy
### GPIO 설치법
  $ sudo apt-get install RPi.GPIO
